# User Account Management Project (Adding, Modifying & Deleting Users)

About

This project demonstrates hands-on experience managing user accounts within an Active Directory environment. The objective was to practice common Technical Support and administrative tasks related to user account creation, modification, permissions management, and account removal.

Project Objectives:
- Create new user accounts in Active Directory
- Modify user account properties and permissions
- Reset passwords and manage account settings
- Disable and delete user accounts

Technologies & Tools Used:
- Windows Server
- Active Directory Users and Computers (ADUC)
- Active Directory Domain Services (AD DS)
- Windows Client Machines
- VirtualBox

Administrative Tasks Performed:
1. Created new user accounts and assigned login credentials
2. Modified user details such as names, passwords, and account permissions
3. Configured password reset and account lockout settings
4. Disabled inactive user accounts
5. Deleted user accounts and verified removal from Active Directory
6. Tested user authentication and login functionality

Key Skills Demonstrated:
- Active Directory user management
- Account administration and security practices
- Password management and resets
- Troubleshooting login and authentication issues
- Windows Server administration

Screenshots included in this repository document:
- Creating new user accounts
- Modifying account properties
- Password reset configuration
- Disabling and deleting user accounts
- Successful authentication and account verification

Outcome:

Successfully managed user lifecycle operations within an Active Directory environment, demonstrating practical experience with common IT Support Specialist and Help Desk administrative responsibilities used in enterprise environments.
